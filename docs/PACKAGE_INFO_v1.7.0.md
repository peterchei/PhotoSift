## Build Commands (Windows, from repo root)

```bat
rem Desktop EXE + installer
build.bat

rem Store assets + MSIX package
python create_store_assets.py
create_store_package.bat
"C:\Program Files (x86)\Windows Kits\10\bin\10.0.26100.0\x64\makeappx.exe" pack /d store_package /p PhotoSift.msix

rem Optional: sign the MSIX
signtool sign /fd SHA256 /a /f your_cert.pfx PhotoSift.msix
```

## What's New in 1.7.0

- New: **Low Resolution Detector** (fifth tool, pure PIL, no AI required)
- Fix: Thumbnail crash in Dark and Blur detectors when image cannot be loaded
- Fix: CLIP model no longer loaded at import time in DuplicateImageIdentifier
- Fix: Deprecated PIL private API `_getexif` replaced with public `getexif`
- Fix: Bare `except:` replaced with `except Exception:` in Dark detector
- New: 27 automated tests for Dark Photo Detection (previously had zero coverage)

## Notes

- `PhotoSift.spec` — no changes required for v1.7.0 (Low Resolution Detector adds no new bundled data)
